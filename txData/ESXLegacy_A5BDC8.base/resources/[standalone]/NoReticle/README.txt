/-/ INSTALLATION \-\
1. Drag and drop the "NoReticle" folder into your server's resource folder.
2. Add "start NoReticle" (without quotations) to your server config to start the script when your server starts.

/-/ PERMISSIONS \-\
Add permissions to your permissions config!

To make the reticle visible:
add_ace group.admin Reticle allow

To make the reticle on the stun gun (taser) visible:
add_ace builtin.everyone ReticleStunGun allow
